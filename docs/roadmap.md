# 路线图与企业级优化建议 / Roadmap

> 面向「博彦科技 · 智能投标 / 方案生成系统」的演进规划。本文档列出从内部测试走向商用所需补足的关键能力。
> English version: [roadmap.en.md](./roadmap.en.md)

## 一、当前已具备的能力（v1.0）

- ✅ 知识库：txt / md / pdf / docx 解析 + 中文友好分块 + 向量化入库
- ✅ 检索：ChromaDB 本地向量库 + 可选 Azure AI Search 混合检索
- ✅ 智能问答 / 标书生成（带来源引用）
- ✅ 多租户隔离、登录、RBAC、审计日志
- ✅ Word / PDF 一键导出（中文字体内嵌）
- ✅ **20 套行业模板**（折叠分类）+ 租户自定义模板
- ✅ 管理端：AI 模型 / Provider / API Key / Mock 切换 / 连接测试 / 系统设置
- ✅ 登录页演示 / 正式部署双模式
- ✅ Docker 镜像、CI、80+ 单元测试

## 二、强烈建议在投入正式使用前补强的能力

### 1. 安全与合规
- [ ] **密码哈希**：当前为简化实现，应统一切换 `bcrypt` / `argon2id`
- [ ] **JWT 刷新机制**：当前 token 不可吊销；建议引入 refresh token + 黑名单
- [ ] **审计日志写入只追加（append-only）**与定期归档
- [ ] **接口防滥用**：限流（IP / 用户级）、CSRF（如启用 Cookie）、CORS 白名单
- [ ] **数据加密**：API Key 在 DB 中应 AES-GCM 加密存储（当前为明文）
- [ ] **PII / 商密脱敏**：上传前自动识别身份证 / 手机号 / 银行卡 / 关键报价等
- [ ] **导出水印**：标书 PDF 自动加用户 + 时间戳水印，防止外泄

### 2. 知识库治理
- [ ] **文档版本管理**：同名文档保留历史版本，可回滚
- [ ] **审核工作流**：上传 → 标注 → 审核 → 发布；防止低质量内容污染
- [ ] **元数据丰富化**：行业、客户、年份、金额、中标率、关键技术栈
- [ ] **重复 / 冲突检测**：相似度过高自动合并提示
- [ ] **图片 / 表格抽取**：PDF 中的表格与架构图单独入库
- [ ] **过期与下架**：基于年份或客户状态自动标记过期

### 3. RAG 与生成质量
- [ ] **混合检索 + 重排序**：BM25 + 向量 + 交叉编码器重排
- [ ] **多查询改写**（HyDE / Multi-Query）以提升召回
- [ ] **章节级生成 + 引用对齐**：每段标注[来源N]，可点击跳转
- [ ] **可控生成**：语气（专业 / 亲和）、长度（精简 / 标准 / 详细）
- [ ] **再生 / 续写 / 针对章节细化**
- [ ] **流式输出（SSE）**：边生成边显示，体验飞跃
- [ ] **提示词版本管理**：prompt 改进可灰度发布与 A/B 测试

### 4. 协作与流程
- [ ] **多人协同编辑**：基于 CRDT / OT 的实时编辑
- [ ] **评审 / 批注**：售前 + 技术 + 商务多角色评审
- [ ] **任务分配**：将 RFP 拆解为章节任务派给团队
- [ ] **版本对比**：标书历次版本 diff
- [ ] **客户标签 / 项目阶段**：与 CRM（Salesforce / 钉钉 / 飞书）打通

### 5. 用户体验
- [ ] **首次登录引导**：4 步带你完成「上传 → 索引 → 问答 → 生成」
- [ ] **拖拽上传**：知识库支持文件拖拽与批量上传
- [ ] **暗色模式**
- [ ] **键盘快捷键**（Ctrl+Enter 提交，Ctrl+K 全站搜索）
- [ ] **历史记录**：最近 N 次问答 / 生成可回放与收藏
- [ ] **引用悬浮卡片**：鼠标悬停显示原文片段，无需跳出

### 6. 可观测与运营
- [ ] **指标采集**：调用次数、token 消耗、平均延迟、错误率
- [ ] **成本面板**：按用户 / 租户 / 模型 统计 token 与金额
- [ ] **告警**：错误率 / 延迟阈值触发企业微信 / 飞书 / 邮件
- [ ] **用户反馈回流**：👍 👎 + 评论，用于改进提示词与检索
- [ ] **A/B 测试框架**：对比不同提示词 / 模型的中标率

### 7. 部署与可扩展性
- [ ] **PostgreSQL 替代 SQLite**（支持高并发与备份）
- [ ] **MinIO / S3 存储**：原始文档独立存储
- [ ] **Redis 缓存**：常见问答缓存
- [ ] **Celery / RQ 异步任务**：大文件解析 / 长生成任务后台执行
- [ ] **Kubernetes Helm Chart**：一键私有化部署
- [ ] **SSO / SAML / OIDC**：与企业域账号集成
- [ ] **多区域部署**与负载均衡

### 8. AI 能力增强
- [ ] **多模态**：图表理解（CLIP / GPT-4V）、招标 PDF 截图解析
- [ ] **Agent 编排**：自动检索竞品报价、自动调用计算/检索工具
- [ ] **企业知识图谱**：项目-客户-技术-人员图谱辅助决策
- [ ] **本地大模型支持**（vLLM / Ollama / 文心 / 通义本地化部署）
- [ ] **微调与 RLHF**：用本企业历史标书做 SFT，更贴合写作风格

## 三、商业化建议

- 多版本：社区版（开源）/ 专业版（私有部署）/ 企业版（含 SSO / 高级安全）
- 与 CRM、ERP、合同管理系统集成（接口标准化）
- 行业模板市场：政务、金融、制造、医疗等可分别售卖
- 培训与服务包：提示词工程培训、知识库初始化服务
